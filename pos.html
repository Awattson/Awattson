研究の背景

近年注目されている画像処理。私たちは画像処理による物体検出を用いて
視覚障碍者の白杖の代理となる物体検出アプリの開発を思いついた。

目的

今回の研究は以上の物体検出アプリの初期段階として、
点字ブロックを検出→バイブレーションで触覚的に位置を知らせる、
androidアプリの開発である。

方法

開発環境

ハードウェア側

  • Windows10
  • raspberrypi3modelB

    • ソフトウェア側

  • OpenCV
  • Kivy
  • Dlib
  • Buildozer

    • 使用言語

  • Python3.9
  • Java

    • 2年次TensorFlowの利用を考えていたが、度重なるエラーにより断念。
    そこで私たちはDlibを利用することにした。
    dlib:C++で書かれた汎用目的のクロスプラットフォームソフトウェアライブラリ
    まずImgLabで学習用画像のラベル付け→xmlファイルで出力
    xmlファイルを使ってDlibで機械学習→データをsvmファイルで出力
    svmファイルを基に点字ブロックを検出するプログラムをPythonで作成
    Buildozerを用いて完成したプログラムファイルをAPKファイルに変換
    →試作版アプリの完成

    結果

    最終的にPCのカメラを通して、点字ブロックを検出することが出来た。
    また、点字ブロックを赤い枠で囲み、プッシュ通知を送ることにも成功。
    この機能をスマートフォン用に一部書き換え、
    BuildozerでAPKファイルに変換をするだけで試作アプリは完成するはずだが
    エラーにより完成には至らなかった。
